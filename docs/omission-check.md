# 원본 설계서 누락 0% 점검표

## A. 작성 지침 반영 여부
- [x] `CLAUDE.md` 는 소개문이 아니라 작업 기준서 성격으로 작성됨
- [x] 누락 우려 시 `docs/` 보조 문서로 분리
- [x] 보조 문서는 `.claude/agents`, `.claude/skills` 가 아니라 `docs/` 에 작성
- [x] LLM 처리 영역과 스크립트 처리 영역 분리 항목이 `CLAUDE.md` 본문에 존재

- [x] 행동 가능한 규칙 중심으로 재구성
