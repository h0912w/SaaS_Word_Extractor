# 파이프라인 스텝별 수행 영역 구분

## 개요

이 문서는 SaaS Word Extractor 파이프라인의 각 스텝별로 **Python 스크립트**가 수행하는 영역과 **Claude 에이전트**가 수행하는 영역을 명확히 구분합니다.

## 상세 설명

### Python 스크립트가 담당하는 영역

1. **파일 I/O 및 데이터 처리**
   - 파일 읽기/쓰기
   - 압축 해제
   - 스트리밍 처리 (대용량 데이터)
   - JSONL/JSON/XLSX/CSV 변환

2. **규칙 기반 처리**
   - 정규화 (소문자화, 특수문자 제거)
   - 규칙 스크리닝 (길이, 문자 비율, 반복 문자)
   - 투표 집계 (알고리즘적 계산)

3. **결과 저장 및 포맷팅**
   - 스키마 검증
   - 메타데이터 작성
   - 사람 검토용 파일 생성

### Claude 에이전트가 담당하는 영역

1. **의미 판정 (Steps 5-7)**
   - SaaS 제목 적합성 판단
   - 레이블 분류 (functional/brandable/ambiguous)
   - 오탐지/오탐지 탐지
   - 경계 케이스 처리

2. **QA 검증 (Step 12)**
   - Recall/Noise/Semantic/Output 감사
   - 최종 품질 판정
   - 개선 사항 제안

## 주요 설계 원칙

1. **Python 스크립트**: 결정론적, 반복적, 대용량 처리
2. **Claude 에이전트**: 의미 판단, 창의적 분석, 경계 케이스 처리
3. **협업 방식**: 스크립트가 데이터를 준비 → 에이전트가 판단 → 스크립트가 결과 저장
